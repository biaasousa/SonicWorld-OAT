# ⚡ Sonic World

O **Sonic World** é uma aplicação web desenvolvida como projeto
acadêmico utilizando **HTML, CSS, JavaScript, Supabase** e o padrão
**MVC (Model-View-Controller)**.

## 📌 Funcionalidades

-   Página Inicial
-   Página Sobre
-   Catálogo de Personagens
-   Página de Contato
-   Login Administrativo
-   Cadastro de Personagens
-   Pesquisa de Personagens
-   Exclusão de Personagens
-   Integração com Supabase

## 🛠 Tecnologias

-   HTML5
-   CSS3
-   JavaScript
-   Supabase
-   MVC

## ▶️ Como executar

1.  Clone este repositório.
2.  Configure o arquivo `js/config.js` com a URL e a chave do Supabase.
3.  Abra o projeto utilizando o Live Server (ou outro servidor local).
